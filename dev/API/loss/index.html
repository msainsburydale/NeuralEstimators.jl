Loss functions

In addition to the standard loss functions provided by Flux (e.g., mae, mse, etc.), NeuralEstimators provides the following loss functions.

NeuralEstimators.kpowerloss ‚Äî Function
kpowerloss(Œ∏ÃÇ, y, k; agg = mean, safeorigin = true, œµ = 0.1)

For k ‚àà (0, ‚àû), the k-th power absolute-distance loss,

\[L(Œ∏ÃÇ, Œ∏) = |Œ∏ÃÇ - Œ∏|·µè,\]

contains the squared-error, absolute-error, and 0-1 loss functions as special cases (the latter obtained in the limit as k ‚Üí 0).

It is Lipschitz continuous iff k = 1, convex iff k ‚â• 1, and strictly convex iff k > 1. It is quasiconvex for all k > 0.

If safeorigin = true, the loss function is modified to avoid pathologies around the origin, so that the resulting loss function behaves similarly to the absolute-error loss in the œµ-interval surrounding the origin.

\[L(Œ∏ÃÇ, Œ∏, q) = (Œ∏ÃÇ - Œ∏)(ùïÄ(Œ∏ÃÇ - Œ∏ > 0) - q),\]

For further discussion on this loss function, see Equation (7) of Cressie, N. (2022), "Decisions, decisions, decisions in an uncertain environment", arXiv:2209.13157.
