Loss functions

In addition to the standard loss functions provided by Flux (e.g., mae, mse, etc.), NeuralEstimators provides the following loss functions for training estimators of type PointEstimator.

NeuralEstimators.tanhloss ‚Äî Function
tanhloss(Œ∏ÃÇ, Œ∏, k; agg = mean, joint = true)

For k > 0, computes the loss function,

\[L(Œ∏ÃÇ, Œ∏) = \textrm{tanh}(|Œ∏ÃÇ - Œ∏|/k),\]

which approximates the 0-1 loss as k ‚Üí 0. Compared with the kpowerloss, which may also be used as a continuous surrogate for the 0-1 loss, the gradient of the tanh loss is bounded as |Œ∏ÃÇ - Œ∏| ‚Üí 0, which can improve numerical stability during training.

If joint = true, the L‚ÇÅ norm is computed over each parameter vector, so that, with k close to zero, the resulting Bayes estimator is the mode of the joint posterior distribution; otherwise, if joint = false, the Bayes estimator is the vector containing the modes of the marginal posterior distributions.

See also kpowerloss.

NeuralEstimators.kpowerloss ‚Äî Function
kpowerloss(Œ∏ÃÇ, Œ∏, k; agg = mean, joint = true, safeorigin = true, œµ = 0.1)

For k > 0, the k-th power absolute-distance loss function,

\[L(Œ∏ÃÇ, Œ∏) = |Œ∏ÃÇ - Œ∏|·µè,\]

contains the squared-error, absolute-error, and 0-1 loss functions as special cases (the latter obtained in the limit as k ‚Üí 0). It is Lipschitz continuous iff k = 1, convex iff k ‚â• 1, and strictly convex iff k > 1: it is quasiconvex for all k > 0.

If joint = true, the L‚ÇÅ norm is computed over each parameter vector, so that, with k close to zero, the resulting Bayes estimator is the mode of the joint posterior distribution; otherwise, if joint = false, the Bayes estimator is the vector containing the modes of the marginal posterior distributions.

If safeorigin = true, the loss function is modified to avoid pathologies around the origin, so that the resulting loss function behaves similarly to the absolute-error loss in the œµ-interval surrounding the origin.

See also tanhloss.

\[S(l, u, Œ∏; Œ±) = (u - l) + 2Œ±‚Åª¬π(l - Œ∏)ùïÄ(Œ∏ < l) + 2Œ±‚Åª¬π(Œ∏ - u)ùïÄ(Œ∏ > u),\]

The method that takes a single value Œ∏ÃÇ assumes that Œ∏ÃÇ is a matrix with $2p$ rows, where $p$ is the number of parameters in the statistical model. Then, the first and second set of $p$ rows will be used as l and u, respectively.
