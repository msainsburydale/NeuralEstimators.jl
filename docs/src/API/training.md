## Training

The function [`train()`](@ref) is used to train a neural estimator.

After training, the risk history and optimiser state can be accessed and inspected using [`loadrisk()`](@ref), [`plotrisk()`](@ref), and [`loadoptimiser()`](@ref).

```@docs
train

loadrisk

plotrisk

loadoptimiser
```
